# AKSIAN Frontend

Next.js e-commerce frontend for the AKSIAN thrift store.

## Setup

```bash
npm install
npm run dev
```

## Project Structure

* `src/app/` - Application routes and layouts
* `src/components/` - UI primitives, sections, and product components
* `src/hooks/` - Custom React hooks
* `src/store/` - Zustand global state
* `public/` - Static assets

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion
* Zustand
* Razorpay

*Note: Detailed documentation exists inside respective folders.*
