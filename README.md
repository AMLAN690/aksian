# AKSIAN Frontend

Next.js e-commerce frontend for the AKSIAN thrift store.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Project Structure

* `frontend/src/app/` - Application routes and layouts
* `frontend/src/components/` - UI primitives, sections, and product components
* `frontend/src/hooks/` - Custom React hooks
* `frontend/src/store/` - Zustand global state
* `frontend/public/` - Static assets

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion
* Zustand
* Razorpay

*Note: Detailed documentation exists inside respective folders.*
